Lab HTML

Attribute

Tag Attribute How does it use for
tag h id ,class ,style tag ที่ใช้กำหนดส่วนหัวตามขนาดตัวอักษร h1-h6
tag p id ,class ,style tag กำหนดข้อความใน paragraph
tag div id ,class ,style tag ที่ใช้จัดการหลายๆ element ให้อยู่ภายใต้ element เดียวกันได้
tag img src, alt, width, height, align, id, class, style tag กำหนดภาพตามที่ได้ระบุแหล่งไฟล์ภาพ และคุณสมบัติของภาพใน attribute
tag a id ,class ,style ,href ,target tag ใช้สำหรับ link ไปยังจุดที่ต้องการ ซึ่งปลายทางอาจเป็น url หรือ html file หรือไฟล์รูปภาพ
tag ul id ,class ,style tag กำหนดรายการ (รายการย่อยแสดงเป็นสัญลักษณ์)
tag ol id ,class ,style tag กำหนดรายการ (รายการย่อยแสดงเป็นลำดับ หรือตัวเลข)
tag table align,border, bordercolor, width bgcolor, cellspacing, cellpadding tag ใช้สำหรับสร้างตารางที่มีการกำหนด row, column, data, border etc. ตามการใช้งานที่ต้องการ
tag form name, method, action, target tag ที่ใช้ในการสร้าง form ตามรูปแบบที่เหมาะสม เพื่อรองรับ input จากผู้ใช้งาน
tag button id ,class ,style ,disabled ,name ,type ,value tag ที่ใช้สำหรับกำหนดปุ่มให้ผู้ใช้งานสามารถ click และตอบสนองข้อมูลบางอย่างปรากฎไปยังผู้ click ได้

Different to 2 Tags

Tags Similarity Differentiation
<hr> & <br> ขึ้นบรรทัดใหม่ hr มีเส้นคั่นกลางก่อนขึ้นบรรทัดใหม่
br ขึ้นบรรทัดใหม่โดยไม่มีเส้นคั่นกลาง
<ul> & <ol> tag ใช้กำหนดรายการย่อย ul กำหนดรายการย่อยแบบประเภทเดียวกัน
ol กำหหนดรายการย่อยแบบมีลำดับ หรือตัวเลข
<select> & <checkbox>
& <radio button>		 เป็นตัวเลือกที่ใช้ใน form select มี list ให้เลือกได้หลายๆ list แต่เลือกได้แค่ 1 choice
checkbox เลือกได้มากว่า 1 choice
ratio มี list จำกัดและเลือกได้แค่ 1 choice
<div> & <span> สามารถจัดการ element ให้อยู่ภายใต้ element เดียวกัน div: หากมีหลาย tag div ติดกันจะขึ้นบรรทัดใหม่
span: หากมีหลาย tag span ติดกันจะไม่ขึ้นบรรทัดใหม่ให้
<script> & <style> tag ที่ใช้กำหนดชุดคำสั่ง script: tag ถูกใช้เพื่อกำหนดชุดคำสั่ง JavaScript
style: tag ใช้ในการกำหนดชุดคำสั่ง CSS
