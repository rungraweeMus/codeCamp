1.CSS Syntax ประกอบด้วย

Selector, Declaration, Property และ Value

ตัวอย่าง: h1 { colour: red, font-size: 10px; }

    Selector: h1

    Declaration: {colour: red, font-size: 10px; }

    Property: colour, font-size

    Value: red, 10px

คำอธิบายการใช้งาน

h2 { font-size: 10px; text-align: right; background-color: red; }

CSS Syntax ข้างต้นได้ระบุ selector เป็น tag h2 อธิบาย Declaration อยู่ภายในเครื่องหมาย {...} ซึ่งประกอบไปด้วย Property 3 ตัว คือ font-size คือการกำหนดขนาดตัวอักษร และได้ระบุค่า value เป็น 10px และ Property คือ text-align สำหรับจัดตำแหน่งของข้อความ ซึ่งได้ระบุค่า value เป็น right คือชิดขวา และ Property คือ background-color ใช้ระบุสีของพื้นหลัง ซึ่งได้ระบุค่า value เป็น red



2.การเขียน CSS มีกี่แบบ อะไรบ้าง

การเขียน CSS แบ่งออกเป็นการเขียน ได้ 3 ประเภท คือ inline, Internal และ External

2.1 การเขียนแบบ inline

คือการเขียนแทรกเข้าไปใน tag html ของแต่ละ element นั้นๆ โดยจะระบุค่า Property เป็น style และจะมีการกำหนด Property และ value ของ css ใน Property ของ style อีกที

ตัวอย่าง

<p style="color: red; font-weight: bold; text-align: center;">ตัวหนังสือหนาสีแดง</p>

2.2 การเขียนแบบ Internal

คือเขียนแยก CSS ออกจาก tag element แต่อยู่ในส่วน tag head ของ file html หน้าเดียวกัน

ตัวอย่าง

<html>
//start tag head
<head>
<style type="text/css">
h1 {
color:peachpuff;
font-weight: 0.9em;
background-color: red;
padding: 5px;
}
</style>
</head>
//end tag head

<body>

<h1>Hello World..</h1>

</body>
</html>

2.3 การเขียนแบบ External

คือการแยกไฟล์ .css ออกมาจากไฟล์ .html เพื่อให้ง่ายต่อการจัดการ code และสามารถ reuse code ใช้ในไฟล์อื่นๆ ได้

ตัวอย่าง

//ไฟล์ style.css

h1 {
color:peachpuff;
font-weight: 0.9em;
background-color: red;
padding: 5px;
}

//ไฟล์ index.html

<html>
//start tag head
<head>
<link rel="stylesheet" href="style.css" >
</head>
//end tag head

<body>

<h1>Hello World..</h1>

</body>
</html>

3.CSS Selector มีกี่แบบ อะไรบ้าง

CSS Selector Declaration
Simple selectors เลือกเอลิเมนต์ด้วยการกำหนดชื่อ element, class หรือ id

div {
font-size: 10px;
}
Combinator Selectors เลือกเอลิเมนต์ด้วยการกำหนดลำดับความสัมพันธ์ tag เช่น div h1{} จะหมายถึง เลือกเอลิเมนต์ h1 ที่ซ้อนอยู่ภายในเอลิเมนต์ div

div h1 {
color: white;
}
Pseudo-class selectors เลือกเอลิเมนต์ตามสถานะ เช่น a:hover หมายถึง เอลิเมนต์ a ที่มีสถานะ hover

a:hover {
background-color: turquoise;
}
Pseudo-elements selectors เลือกกำหนดสไตล์ให้บางส่วนของเอลิเมนต์ เช่น h1::before หมายถึง เลือกกำหนดสไตล์ให้ tag h1 คือ background-color: red ก่อน
หลังจากนั้น จะทำคำสั่งใน h1 {...} คือ background-color: black

h1 {
background-color: black;
}

h1::before {
background-color: red;
}
Attribute selectors เลือกเอลิเมนต์ที่มีแอททริบิวต์ตามที่กำหนด เช่น link[target=”student”] หมายถึง เลือกเอลิเมนต์ link ที่แอททริบิวต์ tartget มีค่าเป็น student

link[target=student] {
font-size: 10px;
}

4.CSS Combinator มี 4 แบบ ดังนี้


CSS Combinator Declaration
space คือ การสั่งการทำงานทั้งหมดของ tag element ทุกตัวไม่ว่าจะเป็น "ลูก หลาน เหลน โหลน" ของ tag ที่อยู่ด้านขวาสุด
ซึ่งอยู่ภายใต้ element ก่อนหน้า โดยการเขียนนั้น ใช้ "ช่องว่าง" ในการสั่งงาน เช่น table tbody tr {...} หมายถึง tr ทั้งหมดที่มีอยู่ใน tbody ซึ่ง tbody อยู่ใน table อีกที
> คือ การสั่งงานเฉพาะ "ลูก" เท่านั้น ไม่นับรวมหลาน หรือเหลน โหลน การเขียนนั้น ใช้ "เครื่องหมายมากกว่า (>)"
เช่น div>p {... } คือ ให้กระทำการใดๆ กับ p ที่เป็น ลูกของ div เท่านั้น ถ้ามี p ซ้อนอยู่ใน p อีกที จะไม่ถูกสั่งการ
+ คือ การสั่งงาน tag ที่อยู่ถัดจาก tag ตัวแรกเพียง tag เดียวเท่านั้น และต้องอยู่ในระดับชั้น element เท่ากัน
การเขียนนั้น ใช้ "เครื่องหมายบวก (+)" เช่น div+div {...} คือ ให้กระทำการใดๆ กับ div ที่อยู่ต่อจาก div ที่ระดับ element เดียวกัน
~ คือ การสั่งงาน tag ที่อยู่ถัดจาก tag ตัวแรกทุกตัว และต้องอยู่ในระดับชั้น element เท่ากัน
การเขียนนั้น ใช้ "เครื่องหมายตัวหนอน (~)" เช่น h1+p {...} คือ ให้กระทำการใดๆ กับ p ที่อยู่หลัง h1 ไม่ว่าจะอยู่ในลำดับที่เท่าไร แต่ p ต้องอยู่ในระดับชั้นเดียวกัน

5. CSS Property และ CSS Value

CSS Property

คือ ชื่อคุณสมบัติที่ต้องการใช้งานตามวัตถุประสงค์ เช่น color ต้องการกำหนดสี , font-size ต้องการกำหนดขนาดตัวอักษร


CSS Value

คือ การกำหนดค่าให้กับ Property เช่น color: red //value=red
และ font-size: 10px //value=10px


ความแตกต่าง

CSS Property ใช้กำหนดคุณสมบัติตามวัตถุประสงค์การใช้งาน(key) ส่วน CSS Value ใช้กำหนดค่า(value)ใน property



6.Property มีกี่กลุ่ม มีอะไรบ้าง


Group Property Declaration
content color, font-size, text-align, font-family
border border-width, border-style, border-color, border-bottom, border-top, border-right
padding padding-top, padding-right, padding-left, padding-bottom
margin margin-top, margin-right, margin-left, margin-bottom
element display, active, width, position, align, flex, grid
